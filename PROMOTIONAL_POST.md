# Plugin: AI Writers Toolkit + PDF Publishing for Joplin

**Transform Joplin into a professional document creation platform**

Are you a writer, consultant, or professional who needs to create polished PDF documents from your notes? The **AI Writers Toolkit + PDF Publishing** plugin transforms Joplin from a note-taking app into a powerful document creation and publishing platform.

## 🎯 Perfect For

- **Writers** creating manuscripts, articles, and reports
- **Consultants** preparing client deliverables and proposals  
- **Professionals** who need to distribute polished PDF documents
- **Anyone** who wants to turn their Joplin notes into beautifully formatted documents

## ✨ Key Features

### 🤖 AI-Powered Writing Assistant
- **Interactive Chat**: Have conversations with ChatGPT directly within Joplin—brainstorm ideas, get feedback, and refine your writing without leaving your notes
- **Grammar & Style Check**: Select any text and get instant grammar, spelling, and stylistic improvements
- **Note-to-Prompt**: Use your current note as context for AI analysis, summarization, or expansion
- **Customizable Persona**: Define how the AI behaves (professional editor, creative co-writer, technical reviewer, etc.)

### 📝 Meeting Notes Made Easy
- **Note Blocks**: Quickly insert formatted note blocks (`note` fenced blocks) to annotate existing documents during meetings
- **Perfect for Document Reviews**: When reviewing an existing document in a meeting, add your notes directly without disrupting the original content
- **Seamless Integration**: Notes appear as formatted blocks that can be styled separately in your PDF output

### 📄 Professional PDF Publishing
Create publication-ready PDFs with sophisticated formatting options:

- **Custom Title Pages**: Set document title, subtitle, author, date, and logo
- **Headers & Footers**: Add custom text with automatic page numbering
- **Flexible Layouts**: Choose page size (Letter/A4) and margin settings
- **Per-Document Settings**: Each note can have its own formatting preferences stored in YAML Front Matter
- **Image Support**: Joplin attachment images render correctly in PDF output
- **Live Preview**: See exactly how your document will look before printing

### 🔄 Two-Way Sync
All PDF settings are stored in YAML Front Matter at the top of your notes:
- Edit settings in the panel → Front Matter updates automatically
- Edit Front Matter directly → Panel syncs on refresh
- Each document retains its own formatting preferences

## 🚀 Use Cases

## 🔧 Installation

1. Open Joplin
2. Go to **Tools > Options > Plugins**
3. Search for **"AI Writers Toolkit + PDF Publishing"**
4. Click **Install**
5. Enter your OpenAI API key in **Tools > Cogitations Plugins > Cogitations Options**

**Version 1.0.2**

Note: The Joplin plugin website may not always reflect the latest version right away, but the plugin repository is kept up to date.

The previous version (1.0.1) was listed as "AI Writing Toolkit + PDF Publishing," so you can search for either "AI Writers Toolkit + PDF Publishing" or "AI Writing Toolkit + PDF Publishing" to find and install the latest version.

## 💡 Quick Start

1. **Set up your API key**: `Tools > Cogitations Plugins > Cogitations Options`
2. **Open the Chat Panel**: `Tools > Cogitations Plugins > Open Chat Panel` (or `Ctrl+Shift+P` → "Toggle AI Writing Toolkit")
3. **Configure PDF Settings**: `Tools > Cogitations Plugins > PDF Publishing`
4. **Add Note Blocks**: Press `Ctrl+Shift+3` (or `Cmd+Shift+3` on macOS) to insert a note block
5. **Generate Preview**: Click "Generate Preview" in the PDF Publishing panel
6. **Print or Save as PDF**: Use the Print button to open the system print dialog
